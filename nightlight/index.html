Introduction

In view of the system and hardware design of PCB layout documentation in information technology is the major term that encompasses all written documents and materials dealing with a PCB layout output and usage. All software and hardware output, whether created by a small team, activity and skills from school: Pamantasan ng lungsod ng muntinlupa, require some related documentations.

The purpose behind this documentation is aim to propose the night light Activity and to introduce how software and hardware prototype developed by the students of CITCS Department in Pamantasan ng lungsod ng muntinlupa.

This article is divided into two (2) parts, In the first part it started by analyzing the PCB layout documentation creation, then follows by prototype and benefits of PCB layout documentation, later talks about documentation tools and finally after the purpose of the document there is topic about documentation standard.

Summary

This summary provides components and steps on how to process the PCB layout and to developed it using software and hardware system and tools. The software tools used to construct the software was flux ai, flux ai is a browser-based electronics design tool with built-in support for modern hardware design methodologies: re-usability, collaboration, and simulation. The components and steps required is listed below.
